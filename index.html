PATCH NOTES

V4.0 - True Multiplayer Architecture

  • Model-View-Controller Restructure: The entire physics engine was decoupled from the rendering loop. The game now runs perfectly via authoritative server state.
  • Local Server Emulator: "Play Offline Sandbox" now boots a hidden LocalServer object that simulates 100% of network traffic and physics, ensuring no frame drops or UI bugs.
  • Server Browser GUI: Added a sleek server list and direct IP connection option (perfect for playit.gg tunnels).
  • Advanced Context AI: Bots dynamically dodge bullets, flee when health is low, and avoid the Alpha Pentagon if they are weak.
  • Predictive Aiming: Hard and Medium bots calculate your velocity to shoot where you are going to be!
  • Level 100 Cap & Omni-Tank: Max level raised to 100. Reach it to evolve into the unstoppable 8-directional Omni-Tank.
  • Kill Economy Fixed: Killing enemies grants 80% of their score but is hard-capped to a maximum of +5 levels per kill.
  • Level 7 Ultimate Perks: Maxing a stat to 7 now grants an ultimate ability (e.g. Smasher, Kiter, Sniper) but locks your other stats to level 3.
